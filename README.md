# EggDualNet

Dual-Branch Network with Spatial GCN for 10-Class Eggplant Disease Classification

## Files
- `EggDualNet_Master.ipynb` — Jupyter notebook with full training and evaluation code
- `eggdualnet_master.py` — All-in-one Python script

## Results
EggDualNet+TTA achieves **97.90% ± 2.02%** accuracy on the 10-class eggplant disease dataset.
